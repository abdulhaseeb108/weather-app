# Weather App 

## Description

A simple weather application that uses the OpenWeatherMap API to show real-time weather and 5-day forecast.

## Features

* Search weather by city
* Auto-detect location using geolocation
* Shows temperature, humidity, wind speed
* 5-day forecast
* Dynamic background based on weather
* Loading & error handling

## Tech Stack

* HTML
* CSS
* JavaScript
* OpenWeatherMap API

## How to Run

1. Clone repo:

```bash
git clone YOUR_REPO_LINK
```

2. Open folder:

```bash
cd weather-app
```

3. Open `index.html` in browser OR use Live Server

## API Used

OpenWeatherMap API:
https://openweathermap.org/api

## Setup

Replace API key in `script.js`:

```js
const apiKey = "YOUR_API_KEY_HERE";
```
